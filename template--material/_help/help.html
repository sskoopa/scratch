Fonts

If you want to host the fonts on your own server instead of Google's CDN (Content Delivery Network), you can either download and the font Roboto Font from web or used font files in folder "fonts".

Otherwise, the fonts will be loaded from Google Font CDN as default to make your web load faster and lower number of connection to your web server.

Fonts used in Template are included in "fonts" folder.

Template

Material is fully responsive HTML5 and CSS3 template. HTML and CSS code is well comented and easy to understand. Group elements are highlighted at the beginning and also at the end by coments.

Basic HTML code structure:

CSS and Sass files for styling the Template can be found in "styles" folder.

Basic CSS code structure:

All JavaScript files, including jQuery plugins can be found in "js" folder. Inside this folder is also folder "vendor" containg offline version of jQuery 2.1.4.

For your own JavaScript code, in "js" folder is prepared file "scripts.js".

Images are included in "images" folder and ready to use.

Every image is optimized to lower the bandwith. All images are styled through CSS, so you don't need to worry when replacing default images with your own.

Template's grid width is 960px and it is fully responsive, ready for every fresolution and device.

Template supports all main broswers - IE10, Firefox, Google Chrome, Safari, Opera, Opera Mini, Android Browser and Chrome for Android.
